## Descripción

Crea una nueva cuenta de usuario y devuelve un token JWT automáticamente.

## Request Body

<ParamField body="username" type="string" required>
  Nombre de usuario único (3-20 caracteres, alfanumérico y guiones bajos)
</ParamField>

<ParamField body="email" type="string" required>
  Email válido y único
</ParamField>

<ParamField body="password" type="string" required>
  Contraseña (mínimo 6 caracteres)
</ParamField>

<ParamField body="display_name" type="string">
  Nombre para mostrar (opcional, por defecto usa username)
</ParamField>

## Response

<ResponseField name="token" type="string">
  Token JWT válido por 24 horas
</ResponseField>

<ResponseField name="user" type="object">
  Información del usuario creado
  
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      ID único del usuario
    </ResponseField>
    
    <ResponseField name="username" type="string">
      Nombre de usuario
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Email del usuario
    </ResponseField>
    
    <ResponseField name="display_name" type="string">
      Nombre para mostrar
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Fecha de creación (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.retrogamehub.games/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "newplayer",
    "email": "newplayer@example.com",
    "password": "SecurePass123!",
    "display_name": "New Player"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.retrogamehub.games/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'newplayer',
    email: 'newplayer@example.com',
    password: 'SecurePass123!',
    display_name: 'New Player'
  })
});

const data = await response.json();
localStorage.setItem('token', data.token);
```

```python Python
import requests

response = requests.post(
    'https://api.retrogamehub.games/auth/register',
    json={
        'username': 'newplayer',
        'email': 'newplayer@example.com',
        'password': 'SecurePass123!',
        'display_name': 'New Player'
    }
)

data = response.json()
token = data['token']
```

## Validaciones

<AccordionGroup>
  <Accordion title="Username">
    - Longitud: 3-20 caracteres
    - Caracteres permitidos: letras, números, guiones bajos
    - Debe ser único en el sistema
    - Case-insensitive (se guarda como minúsculas)
  </Accordion>
  
  <Accordion title="Email">
    - Formato válido: ejemplo@dominio.com
    - Debe ser único en el sistema
    - Se envía email de verificación (futuro)
  </Accordion>
  
  <Accordion title="Password">
    - Mínimo 6 caracteres
    - Se recomienda: mayúsculas, minúsculas, números y símbolos
    - Hasheado con bcrypt antes de guardar
  </Accordion>
  
  <Accordion title="Display Name">
    - Opcional (usa username si no se proporciona)
    - Máximo 50 caracteres
    - Puede contener espacios y caracteres especiales
  </Accordion>
</AccordionGroup>

## Rate Limiting

- **Límite**: 5 registros por hora por IP
- **Protección**: Previene creación masiva de cuentas
- **Status 429**: Demasiadas peticiones

## Seguridad

<Warning>
  La contraseña se hashea con bcrypt (10 salt rounds) antes de almacenarla. Nunca se guarda en texto plano.
</Warning>

<Note>
  El usuario queda automáticamente autenticado después del registro. El token devuelto es válido por 24 horas.
</Note>
