Una plataforma cloud-native para jugar juegos clásicos de DOS usando el emulador js-dos, construida con prácticas modernas de DevOps y arquitectura de microservicios.

## Tecnologías Utilizadas

- **Frontend**: React, js-dos, interfaz web moderna
- **Backend**: Node.js, microservicios REST
- **Base de Datos**: PostgreSQL, Redis para caché
- **Infraestructura**: AWS EKS, Kubernetes, Terraform
- **Monitoreo**: Prometheus, Grafana, alertas automatizadas
- **DevOps**: GitHub Actions, Docker, CI/CD automatizado
