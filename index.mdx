Una plataforma cloud-native para jugar juegos clásicos de DOS usando el emulador js-dos, construida con prácticas modernas de DevOps y arquitectura de microservicios.

## Tecnologías Utilizadas

- **Frontend**: React, js-dos emulator
- **Backend**: Node.js, Express, microservicios
- **Base de Datos**: PostgreSQL, Redis
- **Infraestructura**: AWS EKS, Terraform
- **Monitoreo**: Prometheus, Grafana
- **CI/CD**: GitHub Actions, ArgoCD
- **Autenticación**: JWT, GitHub OAuth
