## Stack Tecnológico

- **Orquestación**: Kubernetes (AWS EKS)
- **Infraestructura como Código**: Terraform
- **Gateway API**: Kong
- **Bases de Datos**: PostgreSQL, Redis
- **Monitoreo**: Prometheus, Grafana, AlertManager
- **CDN**: CloudFront + S3
- **Autenticación**: JWT + GitHub OAuth (monitoreo)
