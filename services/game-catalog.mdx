## Descripción

El servicio de catálogo gestiona el inventario de juegos retro disponibles en la plataforma, incluyendo metadata, imágenes y archivos .jsdos.

- **Integración**: APIs para otros servicios

## Endpoints API

### GET /games
Obtiene lista de todos los juegos disponibles.

```json
{
  "games": [
    {
      "id": 1,
      "title": "DOOM",
      "slug": "doom",
      "description": "Legendary FPS game",
      "year": 1993,
      "genre": "FPS",
      "developer": "id Software",
      "image_url": "/images/doom.jpg",
      "jsdos_file": "/games/doom.jsdos",
      "created_at": "2024-01-01T00:00:00Z"
    }
  ]
}
```

### GET /games/:slug
Obtiene detalles de un juego específico.

### GET /games/search?q=:query
Búsqueda de juegos por nombre.

### POST /games
Agrega un nuevo juego al catálogo.

### PUT /games/:id
Actualiza información de un juego.

### DELETE /games/:id
Elimina un juego del catálogo.

## Base de Datos

### Tabla: games

```sql
CREATE TABLE games (
  id SERIAL PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  slug VARCHAR(255) UNIQUE NOT NULL,
  description TEXT,
  year INTEGER,
  genre VARCHAR(100),
  developer VARCHAR(255),
  image_url VARCHAR(500),
  jsdos_file VARCHAR(500),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Integración con Otros Servicios

- **Servicio de Puntuaciones (3.4)**: Proporciona información de juegos

- **Frontend**: Catálogo visual y búsqueda
